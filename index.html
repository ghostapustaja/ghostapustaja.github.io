LegionOS 1.0

Select an image

Desktop image

The desktop image allows you to try LegionOS without changing your computer at all, and at your option to install it permanently later. This type of image is what most people will want to use. You will need at least 192MiB of RAM to install from this image.

64-bit PC (AMD64) desktop image

Choose this if you have a computer based on the AMD64 or EM64T architecture (e.g., Athlon64, Opteron, EM64T Xeon, Core 2). Choose this if you are at all unsure.

LegionOS can take advantage of the Ubuntu support because it is based on Ubuntu.

If you need help burning the image to disk, see the Image Burning Guide.
